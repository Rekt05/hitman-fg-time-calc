Configure Fast Splitting

Fast splitting currently only works for the listed maps (Trilogy and S1-S3)

Select the amount of digits you need to insert for a map for it to automatically move on to the next map field, ex. if you insert a 2 next to paris you will need to insert a 2 digit number ex. 25 or 59, for 3 digits you can insert ex. 1:01 or 6:59.
